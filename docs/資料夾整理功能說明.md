# 影片自動分割工具 - 資料夾整理功能說明

## 🆕 新功能：自動資料夾整理

### 功能描述
分割完成的影片現在會自動放入以影片名稱命名的專屬資料夾中，讓檔案管理更加整潔有序。

### 資料夾命名規則

#### 單影片模式
- 資料夾名稱：使用影片檔名（不含副檔名）
- 例如：`tennis_match.mp4` → 資料夾名稱為 `tennis_match`

#### 雙影片模式
- 側面影片資料夾：`{影片名稱}_側面`
- 45度影片資料夾：`{影片名稱}_45度`
- 例如：
  - `match_side.mp4` → 資料夾名稱為 `match_side_側面`
  - `match_45deg.mp4` → 資料夾名稱為 `match_45deg_45度`

#### 重複名稱處理
如果已存在相同名稱的資料夾，系統會自動添加編號：
- 第一次：`tennis_match`
- 第二次：`tennis_match_1`
- 第三次：`tennis_match_2`
- 以此類推...

### 資料夾結構示例

#### 單影片模式輸出結構：
```
video_segments_output/
└── tennis_match/
    ├── tennis_match_segment_01.mp4
    ├── tennis_match_segment_02.mp4
    ├── tennis_match_segment_03.mp4
    └── segmentation_info_側面.json
```

#### 雙影片模式輸出結構：
```
video_segments_output/
├── match_side_側面/
│   ├── match_side_segment_01.mp4
│   ├── match_side_segment_02.mp4
│   └── segmentation_info_側面.json
└── match_45deg_45度/
    ├── match_45deg_segment_01.mp4
    ├── match_45deg_segment_02.mp4
    └── segmentation_info_45度.json
```

### 使用方式
1. **選擇影片檔案**：照常選擇要分割的影片
2. **設定輸出路徑**：設定主要輸出資料夾位置
3. **執行分割**：系統會自動在主要輸出資料夾內創建影片專屬子資料夾
4. **完成後**：可選擇直接開啟影片專屬資料夾或主要輸出資料夾

### 優點
- ✅ 檔案組織更清楚
- ✅ 避免不同影片的片段混在一起
- ✅ 方便後續管理和查找
- ✅ 支援多次分割同一影片而不覆蓋之前的結果
- ✅ 雙影片模式下各角度影片分開存放

### 注意事項
- 資料夾會以影片檔名（不含副檔名）作為基礎名稱
- 如果影片檔名包含特殊字符，建議先重新命名影片檔案
- 系統會自動處理資料夾名稱衝突，確保不會覆蓋現有檔案
